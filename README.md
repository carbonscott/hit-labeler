# Game of Hit

A GUI for labeling SPI hits.


## Install with `pip`

```
pip install git+https://github.com/carbonscott/hit-labeler --upgrade --user
```


## Dependency

```
pyqtgraph
numpy
```

## Shortcuts

- `N`: next query image
- `P`: prev query image
- `L`: label an image
- `G`: go to a specific image
